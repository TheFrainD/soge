helper for R (row vector) * M (matrix) * C (column vector)

rmc stands for Row * Matrix * Column

the result is scalar because R * M = Matrix1x2 (row vector), then Matrix1x2 * Vec2 (column vector) = Matrix1x1 (Scalar)
