CGLM_INLINE mat4s glms_mat4_mulN ( mat4s *__restrict  matrices[],
uint32_t  len 
)

mupliply N mat4 matrices and store result in dest

this function lets you multiply multiple (more than two or more...) matrices

multiplication will be done in loop, this may reduce instructions size but if len is too small then compiler may unroll whole loop, usage:

mat m1, m2, m3, m4, res;
res = glm_mat4_mulN((mat4 *[]){&m1, &m2, &m3, &m4}, 4);
Warning
matrices parameter is pointer array not mat4 array!
Parameters
[in]matricesmat4 * array
[in]lenmatrices count
Returns
result matrix
